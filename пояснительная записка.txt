Kirilization 3 - аналог популярной игры civilization 3. Kirilization 3 предназначена для игры нескольких людей на одном устройстве.
Когда запускается игра, на экране появляется выбор команд которые будут участвовать в игре.

При запуске основного экрана игры, видна игровая карта, которая каждый раз генерируется случайным образом. Перемещать камеру по игровому полю можно кнопка W, A, S, D. 
Справа снизу находится надпись которая показывает какая команда сейчас делает ход. Чуть ниже этой надписи находится кнопка для передачи хода следующему игроку.
У каждой команды в начале игры есть только один юнит - поселенец. При нажатии мышью по юниту, внизу отображается информация о нем - Имя, Здоровье, Команда и очки перемещения.
Если был нажат юнит команды которая сейчас делает ход, то при нажатии на рядом лежащуюю клетку, совершится перемещение юнита туда. Перемещение совершается только в случае если хватает очков перемещения.
Игровое поле поделено на клетки разных биомов. У каждого биома есть количество очков которое требуется для перемещения на данную клетку. Самые легкие для перемещения биомы это равнины и холмы, самые сложные это вода, тундра и горы.
На некоторых клетах присутствуют ресурсы. Они отображаются маленьким квадратиком. У каждого ресурса есть свой тип: красный - стратегический, желтый - Ценные, синие - бонусные.
Если выбрать поселенца команды которая сейчас ходит и нажать на пробел, на клетке где стоял поселенец образуется город (ресурс который был на клетке удаляется). При нажатии на город появляется информация о нем.
Имя, население, производительность, пища, золото. Последние три поля зависят от соседних с городом клеток и ресурсов. Сверху экрана информации о города есть две кнопки - левая для постройки зданий и правая для производства юнитов.
При нажатии на одну из кнопок появляется меню возможных вариантов для производства зданий или юнитов. При выборе какого то здания или юнита, на основном экране в поле очередь появляется это здание или юнит и число которое отображает
необходимое количество ходов для производства этого здания или юнита. Здания приносят городу очки производства, еду, золото, науку.
Наука - невидимое число которое каждый ход увеличивается на общее число науки всех городов на карте. Как только число науки превысило необходимую планку, дается одно очко технологий.
Слева сверху есть синяя кнопка, нажав на которую, откроется меню технологий, каждую технологию можно открыть за одно очко технологий. Эти технологии открывают новых юнитов для производства и различные бонусы.
